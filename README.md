# Yamini

Yamini is a stack based RISC type virtual machine with a bytecode IR to plug your language front-ends into.  

## Steps to run

1. To run a binary:

```bash
user@programmer~:$ cargo run <filepath>
```

2. To run the tests:

```bash
user@programmer~:$ cargo test
```
